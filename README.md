# GameLog
High-Performance Portable Log Window for unity

## Usage
* Just drag & drop GameLog-Window prefab into your scene
```c#
GameLog.Log("Kiarash Parvizi");
GameLog.Log("Running...", UnityEngine.Color.cyan);
```
